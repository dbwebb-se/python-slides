Truthiness, Try-Except och Flow

Truthiness

Truthiness-uppgift

Skriv ett program som validerar lösenord.

Ett giltig lösenord uppfyller följande krav:

  • Innehåller minst ett tecken.
  • Innehåller max 10 tecken.
  • Innehåller minst ett heltal. Tips använd funktionen isdigit()

Tips att fokusera på ett krav i taget och inte att försöka jämföra/validera allt på en gång.

Try-Except

  • Syntax error
  • Runtime error/Exceptions
  • Semantic/Logic error
try:
    # try block
    STATEMENTS
except EXCEPTION:
    # If exception run this code
    STATEMENTS
else:
    # if no exception run this code
    STATEMENTS
finally:
    # Always run this
    STATEMENTS

Skriv ett program som tar emot input från användaren. Beroende på inputens data typ gör en av två saker:

En sträng: Konkatenera ihop med tidigare stränger.

Let's Code Uppgift

Skriv ett program som tar emot tre stycken input.

Först ska programmet ta emot ett ord.

Sedan ska programmet i två omgångar ta emot ett heltal.

Skriv sedan ut med hjälp av en for-loop delsträngen mellan dina två siffror.

Använd if-satser för att kolla att dina två heltal hamnar inom strängens längd. Använd try-except för att kolla att vi får in rätt värden.

Praktisk problemlösning

Skriv ett program som ska hitta vilken index position en substräng startar och slutar på i en större sträng. Ex substrängen "j e" börjar på index 2 och slutar på 4 i "Hej Emil".

I din lösning får du inte använda några inbyggda funktioner förutom range(), enumerate() och lower().

Kielbasa ball tip alcatra spare ribs chuck fatback shankle hambutt hock pig pork chop strip steak landjaeger. Hamburger porchetta frankfurter buffalo swine tongue."
"hamburger"
