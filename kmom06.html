Dokumentation

Dictionaries uppgift

Skriv ett program som frågar användaren efter ett namn och en skostorlek, tills användaren skriver done.

Programmet skriver sedan ut en sorterad lista med 'namn: skostorlek' och avslutar med medelvärdet av skostorlekarna.

Blandade datatyper

Vi utgår från vår courses-dict.

courses = {
    "python" : "Programmering och problemlösning med Python",
    "htmlphp" : "Webbteknologier"
}

Uppgift

Fortsätt från ditt tidigare program med namn och skostorlek. Istället för skostorleken spara en dict med skostorlek och tröjstorlek samt en lista med tuples som representerar de 3 senaste GPS-koordinater där en person har varit.

Exempel:

persons = {
    "Emil" : {
        "skostorlek": 44,
        "tröjstorlek": "L",
        "koordinater": [
            (40.712, -74.005),
            (28.358, -14.053),
            (55.927, 12.300)
        ]
    }
}
