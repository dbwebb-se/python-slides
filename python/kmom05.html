Tuples uppgift

Skriv ett program där man kan söka efter filmer. Det ska gå att söka på namn, år utgiven och regisör för filmerna. Alla filmer som matchar sökningen ska visas och vara sorterade i bokstavsordning på filmens namn. Spara data om en film tuple och spara alla tuples i en lista.
Exempel: Om filmerna "Scott Pilgrim vs. the world" och "Baby Driver" finns i listan och man söker efter "Edgar Wright" ska båda komma upp som resultat och "Baby Driver" visas överst.

I example/lectures/2020/kmom05/movies.txt finns en tuple med filmer som ni kan använda för uppgiften.

Dictionaries

dict

I andra programspråk kallas dictionaries ofta Associative Arrays, Map eller Hash(map).

Dictionaries

Gemensamt är att det alltid finns ett nyckel-värde par.

courses = {
                "python" : "Programmering och problemlösning med Python",
                "webtec" : "Webbteknologier"
            }

Dictionaries egenskaper

  • Mutable
  • Sorterade (från python 3.6)
  • Tillåter inte dubbletter
  • Nyckeln immutable
  • Värdet - vilken datatyp som helst

Dictionaries uppgift

Skriv ett program som frågar användaren efter ett namn och en skostorlek, och spara i en dictionary, tills användaren skriver done.

Programmet skriver sedan ut en sorterad lista, på sko storlek, med 'namn: skostorlek' och avslutar med medelvärdet av skostorlekarna.

Lyft exception

Lyft ett exception när koden eventuellt kommer att stöta på ett scenario där exekveringen inte kan fortsätta.

Genom att lyfta ett exception, ett planerat avbrott i koden, så kommer det att tillåta andra delar av koden att fungera korrekt, så att exekveringen kan fortsätta.

Lyft exception

Exempel på hur fel kan lyftas:

raise IndexError("List index out of range!")

eller

raise ValueError # short for raise ValueError()

Lyft exception uppgift

Fortsätt på uppgiften med skorna. Du göra en funktion check_size(shoe_size) som omvandlar shoe_size till en int och lyfter felet ValueError om:
shoe_size är större än 48 med texten "The shoe size is too large"
shoe_size är mindre än 32 med texten "The shoe size is too small"
